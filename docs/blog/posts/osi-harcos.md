# ⚔️ Új mesterség: Ősi Harcos

A birodalom régésze újabb elfeledett tudást hozott napvilágra — bekerült az **Ősi Harcos** mesterség! Ha te is a harc mestere vagy, itt az idő bizonyítani.

<!-- more -->

## Hogyan szerezheted meg?

Az Ősi Harcos cím eléréséhez össze kell gyűjtened az **összes, túlélő módban megszerezhető mob fejet**. Ez nem kis feladat — a legtöbb fejhez egy **Charged Creeper** robbanására lesz szükséged!

### Szükséges fejek

| Mob fej | Megszerzés módja |
|---|---|
| 🧟 Zombie fej | Charged Creeper robbanás |
| 💀 Csontváz koponya | Charged Creeper robbanás |
| 🟢 Creeper fej | Charged Creeper robbanás |
| 🐷 Piglin fej | Charged Creeper robbanás (Nether) |
| 🖤 Wither Skeleton koponya | Wither Skeleton loot (Nether Fortress) |
| 🐉 Ender Sárkány fej | End Ship-eken található |

!!! tip "Tipp"
    A Charged Creeperhez villámcsapás kell — használj **Tridentet Channeling** bűvöléssel viharban, vagy várj természetes villámcsapásra!

---

## Mit kapsz cserébe?

Az Ősi Harcos cím birtokosaként hozzáférsz az ősi bűvöléshez, amellyel **a fegyvereid bűvöléseit az eredeti maximum fölé emelheted**. Az alábbi enchantok fejleszthetők tovább:

| Bűvölés | Vanilla max | Emelhető |
|---|:---:|:---:|
| Élesség (Sharpness) | V (5) | ✓ |
| Erő (Power — íj) | V (5) | ✓ |
| Szikrából pattant (Fire Aspect) | II (2) | ✓ |
| Hátralökés (Knockback) | II (2) | ✓ |
| Zsákmányolás (Looting) | III (3) | ✓ |
| Suhintás (Sweeping Edge) | III (3) | ✓ |

!!! important "Fontos"
    Az ősi bűvölés **[gyémánt](../../gazdasag.md), lazurit és xp szint költséggel** jár, és az ár szintenként egyre nő. Gondolkodj befektetésként!
